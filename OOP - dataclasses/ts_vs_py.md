# 🔄 TypeScript vs. Python OOP Reference

This guide provides a quick side-by-side mapping of common Object-Oriented Programming (OOP) concepts between TypeScript and Python.

---

## 🗺️ Concept Mapping

| TypeScript Concept | Python OOP Equivalent |
| :--- | :--- |
| `interface` | `Protocol` (structural typing) or `ABC` (for behavior) |
| `class` | `class` |
| `constructor` | `__init__()` |
| `toString()` | `__str__()` |
| `console.log(obj)` | `__repr__()` |
| `===` (custom object equality) | `__eq__()` |
| `Map`/`Set` key hashing | `__hash__()` |
| `readonly` | `@dataclass(frozen=True)` |
| `abstract class` | `ABC` + `@abstractmethod` |
